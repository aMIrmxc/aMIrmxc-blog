 🚀 Next.js 15 منتشر شد!

مهم‌ترین آپدیت Next.js بالاخره اومده و این نسخه واقعاً ارزش توجه داره!

چرا این آپدیت مهمه؟

🔹 بزرگ‌ترین تغییر: دیگه fetch به‌صورت پیش‌فرض cache نمی‌شه! کنترل کامل دست توسعه‌دهندس
🔹 Async APIs: headers، cookies و params حالا async شدن
🔹 React 19: پشتیبانی کامل از hook جدید `use`
🔹 Turbopack: تا ۱۰ برابر سریع‌تر startup
🔹 API جدید after(): اجرای کد بعد از ارسال response

و خبر خوب اینکه:
مهاجرت خیلی راحته! با یه دستور:

```bash
npx @next/codemod@canary upgrade latest
```

همه چیز خودکار انجام می‌شه.

توی وبلاگم، تمام breaking change ها و تغییرات رو با مثال‌های عملی توضیح دادم. 👇
