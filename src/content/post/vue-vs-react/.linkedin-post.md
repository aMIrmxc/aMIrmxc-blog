🔹 ساختار کامپوننت‌ها (JSX vs فایل‌های .vue)
🔹 مدیریت State (useState vs ref/reactive)
🔹 شرط‌ها و حلقه‌ها (map/&& vs v-if/v-for)
🔹 Props (destructuring vs defineProps)
🔹 lifecycle متدها (useEffect vs onMou      nted/watch)
🔹 Computed values (useMemo vs computed)
🔹 فرم‌ها و Two-way binding (value+onChange vs v-model)
🔹 Children vs Slots
🔹 Context API vs Provide/Inject
🔹 Custom Hooks vs Composables
🔹 useRef vs Template Refs
🔹 Redux/Zustand vs Pinia
🔹 و یک پروژه To-Do List واقعی 🎯
