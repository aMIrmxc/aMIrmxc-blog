#  برسی مهمترین دستورات لینوکس - از مبتدی تا حرفه‌ای


![alt text](img.png)



لینوکس 🐧  قدرتمندترین و از محبوب‌ترین سیستم‌عامل‌های جهان است که در سرورها، کامپیوترهای شخصی، دستگاه‌های موبایل و ... استفاده می‌شود. یکی از ویژگی‌های بارز لینوکس، دسترسی مستقیم رابط خط فرمان یا Terminal  به هسته سیستم عامل است که به کاربران امکان می‌دهد کنترل کامل بر سیستم از طریق ترمینال داشته باشند. در این بلاگ پست، با مهم‌ترین دستورات لینوکس آشنا خواهیم شد .

یادگیری دستورات لینوکس نه تنها مهارتی ضروری برای مدیران سیستم و توسعه‌دهندگان است، بلکه درک عمیقی از نحوه عملکرد سیستم‌عامل به شما می‌دهد. برخلاف رابط گرافیکی، خط فرمان امکان اتوماسیون وظایف، مدیریت از راه دور و دسترسی به قابلیت‌های پیشرفته را فراهم می‌کند که در محیط گرافیکی بسیار دشوار است.

برای شروع آماده اید؟

## ۱. ورود با دسترسی Sudo (یا super user do )

حساب کاربری root در لینوکس بالاترین سطح دسترسی را دارد و می‌تواند هر تغییری در سیستم ایجاد کند. برای فعالیت‌های مدیریتی و پیکربندی سیستم، اغلب نیاز به دسترسی root دارید. در این بخش نحوه ورود به حساب root و تعیین رمز عبور برای آن را یاد می‌گیرید.

### تعیین رمز برای حساب Root

برای تعیین یا تغییر رمز عبور حساب root، ابتدا باید با دستور sudo وارد محیط root شوید و سپس دستور passwd را اجرا کنید. این کار معمولاً اولین قدم پس از نصب لینوکس است، زیرا به صورت پیش‌فرض حساب root رمز عبور ندارد.

```bash
sudo -i passwd
```

پس از اجرای این دستور، سیستم از شما می‌خواهد رمز عبور جدید را دو بار وارد کنید. توجه داشته باشید که هنگام تایپ رمز عبور، هیچ کاراکتری روی صفحه نمایش داده نمی‌شود که این یک ویژگی امنیتی است.

### ورود به محیط Sudo (یا super user do )

برای ورود به محیط Sudo و دسترسی به تمام امکانات مدیریتی، از دستور زیر استفاده کنید. این دستور به شما این امکان را میدهد که  بدون تایپ sudo  ( که همه ی محدودیت های امنیتی را کنار میزند ) اقدام به اجرای دستورات مدیریتی حساس نمایید.

```bash
sudo -i
```

پس از وارد کردن این دستور و تایپ رمز عبور کاربر فعلی، خط فرمان به `#` تغییر می‌کند که نشان‌دهنده دسترسی root است. در این حالت باید بسیار مراقب باشید، زیرا هر دستور اشتباه می‌تواند منجر به از بین رفتن داده‌ها یا خرابی سیستم شود.
تنها دستورات حساس برای اجرا نیاز به نوشتن sudo در ابتدا آن دستور یا رفتن به محیط sudo با زدن دستور بالا را دارند.

## ۲. دستور clear - پاک کردن صفحه ترمینال

دستور clear یکی از ساده‌ترین و در عین حال پرکاربردترین دستورات لینوکس است. پس از کار طولانی با ترمینال، صفحه پر از خروجی‌های مختلف می‌شود که می‌تواند باعث سردرگمی شود. این دستور صفحه ترمینال را پاک کرده و یک محیط تمیز برای کار فراهم می‌کند.

```bash
clear
```

> 💡 **نکته کاربردی:** می‌توانید از کلید میانبر `Ctrl+L` نیز برای پاک کردن صفحه استفاده کنید که سریع‌تر از تایپ دستور clear است. این کلید میانبر در اکثر ترمینال‌ها کار می‌کند و بدون نیاز به تایپ هیچ دستوری، صفحه را پاک می‌کند.

## ۳. دستور pwd - نمایش مسیر فعلی

دستور `pwd` مخفف **Print Working Directory** است و مسیر کامل دایرکتوری فعلی که در آن قرار دارید را نمایش می‌دهد. این دستور به ویژه زمانی مفید است که در دایرکتوری‌های تو در تو حرکت می‌کنید و می‌خواهید بدانید دقیقاً کجا هستید.

```bash
pwd
```

خروجی این دستور یک مسیر مطلق است که از ریشه سیستم فایل (`/`) شروع می‌شود. برای مثال، اگر در دایرکتوری خانگی کاربر باشید، خروجی چیزی شبیه به `/home/username` خواهد بود. مسیرهای مطلق هم از ریشه شروع می‌شوند و مسیر کاملی برای رسیدن به موقعیت فعلی ارائه می‌دهند.

---

## ۴. دستور cd - تغییر دایرکتوری

دستور `cd` مخفف **Change Directory** است و اصلی‌ترین ابزار برای پیمایش در سیستم فایل لینوکس محسوب می‌شود. با استفاده از این دستور می‌توانید بین دایرکتوری‌های مختلف حرکت کنید و به محل مورد نظر خود دسترسی پیدا کنید.

### رفتن به دایرکتوری مشخص

برای رفتن به یک دایرکتوری خاص، کافی است مسیر آن را بعد از دستور `cd` وارد کنید. مسیر می‌تواند مطلق (از ریشه شروع شود) یا نسبی (از دایرکتوری فعلی شروع شود) باشد.

```bash
cd Your_New_Directory_Path
```

در این دستور، `Your_New_Directory_Path` می‌تواند مسیر نسبی یا مطلق باشد. مسیرهای مطلق با `/` شروع می‌شوند.

### رفتن به دایرکتوری ریشه (Root)

برای رفتن به دایرکتوری ریشه سیستم که بالاترین سطح سلسله‌مراتب فایل‌هاست، از دستور زیر استفاده کنید:

```bash
cd /
```

### رفتن به دایرکتوری خانه (Home)

برای بازگشت سریع به دایرکتوری خانگی خود، می‌توانید از یکی از دو دستور زیر استفاده کنید. کاراکتر `~` در لینوکس نشان‌دهنده مسیر خانه کاربر فعلی است.

```bash
cd ~
cd
```

### رفتن به دایرکتوری والد

برای رفتن به دایرکتوری والد (یک سطح بالاتر)، از دو نقطه استفاده کنید:

```bash
cd ..
```

### رفتن به دایرکتوری قبلی

برای بازگشت به آخرین دایرکتوری که در آن بوده‌اید، از کاراکتر خط تیره استفاده کنید:

```bash
cd -
```

استفاده از این دستور باعث می‌شود مسیر قبلی شما ذخیره شود و بتوانید به سرعت بین دو دایرکتوری جابجا شوید. همچنین مسیر دایرکتوری قبلی نمایش داده می‌شود.

## ۵. دستور ls - نمایش محتویات دایرکتوری

دستور `ls` یکی از پرکاربردترین دستورات لینوکس است که محتویات یک دایرکتوری را فهرست می‌کند. این دستور اطلاعات فایل‌ها و زیردایرکتوری‌ها را نمایش می‌دهد و با گزینه‌های مختلف، خروجی‌های متنوعی ارائه می‌دهد.

### نمایش محتویات دایرکتوری

برای دیدن محتویات دایرکتوری فعلی یا یک دایرکتوری خاص:

```bash
ls                  # نمایش محتویات دایرکتوری فعلی
ls path             # نمایش محتویات مسیر مشخص شده
```

### نمایش محتویات دایرکتوری والد

```bash
ls ..
```

### نمایش زیردایرکتوری‌ها به صورت بازگشتی

```bash
ls *
```

### نمایش جزئیات کامل

برای نمایش جزئیات کامل فایل‌ها شامل مجوزها، مالک، گروه، اندازه و تاریخ اصلاح:

```bash
ls -l
```

خروجی این دستور شامل اطلاعاتی است که هر ستون آن معنای خاصی دارد:

| ستون | توضیحات |
|------|---------|
| ستون اول | مجوزهای فایل (مثلاً `-rwxr-xr-x`) |
| ستون دوم | تعداد لینک‌ها |
| ستون سوم | نام مالک |
| ستون چهارم | نام گروه |
| ستون پنجم | اندازه فایل به بایت |
| ستون ششم | تاریخ و ساعت آخرین تغییر |
| ستون هفتم | نام فایل |

## ۶. دستور mkdir - ساخت دایرکتوری جدید

دستور `mkdir` مخفف **Make Directory** است و برای ایجاد دایرکتوری جدید استفاده می‌شود. این دستور امکانات متنوعی برای ساخت دایرکتوری‌های ساده، چندگانه و تو در تو ارائه می‌دهد.

### ساخت دایرکتوری ساده

```bash
mkdir newDirectoryName                      # ساخت در مسیر فعلی
mkdir PATH/newDirectoryName                 # ساخت در مسیر مشخص
```

### ساخت چند دایرکتوری همزمان

برای ایجاد چند دایرکتوری به صورت همزمان، از آکولاد استفاده کنید. **توجه:** بین نام دایرکتوری‌ها و کاما نباید فاصله باشد.

```bash
mkdir {dir1,dir2,dir3}
```

### اعطای مجوز هنگام ساخت

```bash
mkdir -m a=rwx directory_name
```

در این مثال، `a=rwx` به معنای اعطای مجوز خواندن، نوشتن و اجرا به همه کاربران است.

### ساخت دایرکتوری‌های تو در تو

```bash
mkdir -p directory1/directory2
```

این دستور اگر `directory1` وجود نداشته باشد، ابتدا آن را ایجاد کرده و سپس `directory2` را داخل آن می‌سازد.

## ۷. دستور rmdir - حذف دایرکتوری

دستور `rmdir` مخفف **Remove Directory** است و برای حذف دایرکتوری‌ها استفاده می‌شود. این دستور به صورت پیش‌فرض فقط دایرکتوری‌های خالی را حذف می‌کند.

### حذف دایرکتوری خالی

```bash
rmdir directory_name
```

### حذف دایرکتوری‌های تو در تو

```bash
rmdir -p directory_name
```

### نادیده گرفتن خطا در صورت خالی نبودن

```bash
rmdir --ignore-fail-on-non-empty
```

> ⚠️ **نکته مهم:** این دستور همچنان دایرکتوری غیرخالی را حذف نمی‌کند، فقط خطا را نمایش نمی‌دهد. برای حذف دایرکتوری غیرخالی باید از `rm -r` استفاده کنید که باید با احتیاط به کار رود.

## ۸. دستور man - راهنمای دستورات

دستور `man` مخفف **Manual** است و به شما امکان دسترسی به مستندات کامل هر دستور لینوکس را می‌دهد. این یکی از مهم‌ترین ابزارها برای یادگیری و مرور دستورات است.

```bash
man Command_Name
```

برای مثال، `man ls` راهنمای کامل دستور `ls` را نمایش می‌دهد.

### کلیدهای میانبر در محیط man

| کلید | عملکرد |
|------|--------|
| `Space` | رفتن به صفحه بعد |
| `b` | برگشت به صفحه قبل |
| `q` | خروج از راهنما |
| `/` + کلمه | جستجو در متن |

### بخش‌های راهنمای man

- **NAME**: نام و توضیح کوتاه
- **SYNOPSIS**: نحوه استفاده
- **DESCRIPTION**: توضیحات کامل
- **OPTIONS**: گزینه‌های موجود
- **EXAMPLES**: مثال‌ها
- **SEE ALSO**: دستورات مرتبط

## ۹. دستور cp - کپی فایل

دستور `cp` مخفف **Copy** است و برای کپی کردن فایل‌ها و دایرکتوری‌ها استفاده می‌شود.

### کپی محتوا در فایل جدید

```bash
cp Source_File Destination_File
```

### کپی فایل در همان مسیر

```bash
cp Original_File Copy_File
```

### کپی فایل به دایرکتوری دیگر

```bash
cp Source_File Destination_Directory
```

### گزینه‌های کاربردی

| گزینه | توضیحات |
|-------|---------|
| `-r` | کپی بازگشتی دایرکتوری‌ها |
| `-i` | تأیید قبل از جایگزینی |
| `-p` | حفظ مجوزها و زمان‌های فایل |
| `-v` | نمایش جزئیات عملیات |

## ۱۰. دستور grep - جستجو در فایل

دستور `grep` یکی از قدرتمندترین ابزارهای جستجو در لینوکس است. این دستور به شما امکان می‌دهد یک الگوی متنی را در یک یا چند فایل جستجو کنید.

```bash
grep [عبارت مورد نظر] [آدرس و فایل]
```

### مثال کاربردی

```bash
grep error /var/log/syslog
```

این دستور تمام خطوط حاوی کلمه `error` در فایل لاگ سیستم را نمایش می‌دهد.

### گزینه‌های پرکاربرد

| گزینه | توضیحات |
|-------|---------|
| `-i` | جستجوی بدون حساسیت به حروف بزرگ و کوچک |
| `-r` | جستجوی بازگشتی در دایرکتوری‌ها |
| `-n` | نمایش شماره خط |
| `-c` | شمارش تعداد تطابق‌ها |
| `-v` | نمایش خطوط بدون تطابق |

---

## ۱۱. دستور cat - نمایش و ادغام فایل

دستور `cat` یکی از چندکاره‌ترین دستورات لینوکس است که برای نمایش محتوای فایل، ایجاد فایل جدید، کپی محتوا و ادغام چند فایل استفاده می‌شود.

### نمایش محتوای فایل

```bash
cat file_name                    # نمایش در همان مسیر
cat PATH/file_name               # نمایش فایل در مسیر مشخص
```

### نمایش محتوای چند فایل

```bash
cat file1 file2 file3
```

### ایجاد فایل جدید

```bash
cat > File_Name
```

پس از تایپ دستور، متن خود را وارد کرده و با `Ctrl+D` خروج شوید.

### کپی محتوا به فایل دیگر

```bash
cat [original_file] > [destination_file]
```

### اضافه کردن به انتهای فایل

```bash
cat file1 >> file2
```

### ادغام چند فایل

```bash
cat file1 file2 file3 > merged_file
```

## ۱۲. دستور chmod - تغییر مجوزها

دستور `chmod` مخفف **Change Mode** است و برای تغییر مجوزهای دسترسی به فایل‌ها و دایرکتوری‌ها استفاده می‌شود.

در لینوکس، هر فایل سه سطح دسترسی دارد:
- **Owner (u)**: مالک فایل
- **Group (g)**: گروه مالک
- **Others (o)**: سایر کاربران

هر سطح می‌تواند مجوزهای زیر را داشته باشد:
- **r (read)**: خواندن
- **w (write)**: نوشتن
- **x (execute)**: اجرا

### مجوزها به صورت نمادی

```bash
chmod u=rwx,g=rw,o=r File_Name
```

### مجوزها به صورت عددی

```bash
chmod 755 File_Name
```

### جدول مجوزهای عددی و نمادی

| مجوز عددی | مجوز نمادی | توضیحات |
|-----------|------------|---------|
| 0 | `---` | بدون هیچ مجوزی |
| 1 | `--x` | فقط مجوز اجرا |
| 2 | `-w-` | فقط مجوز نوشتن |
| 3 | `-wx` | مجوز نوشتن و اجرا |
| 4 | `r--` | فقط مجوز خواندن |
| 5 | `r-x` | مجوز خواندن و اجرا |
| 6 | `rw-` | مجوز خواندن و نوشتن |
| 7 | `rwx` | همه مجوزها (کامل) |

> 💡 **نکته:** در مثال `chmod 755`، عدد 7 برای مالک (4+2+1 = خواندن + نوشتن + اجرا)، عدد 5 برای گروه (4+1 = خواندن + اجرا) و عدد 5 برای دیگران (4+1 = خواندن + اجرا) اعمال می‌شود.

## سخن آخر

در این مقاله با مهم‌ترین دستورات لینوکس آشنا شدید که پایه و اساس کار با خط فرمان این سیستم‌عامل قدرتمند را تشکیل می‌دهند. این دستورات روزانه توسط میلیون‌ها کاربر، مدیر سیستم و توسعه‌دهنده در سراسر جهان استفاده می‌شوند و تسلط بر آن‌ها اولین قدم برای تبدیل شدن به یک کاربر حرفه‌ای لینوکس است.

پیشنهاد می‌کنیم این دستورات را به صورت عملی تمرین کنید و از دستور `man` برای یادگیری گزینه‌های بیشتر هر دستور استفاده نمایید. لینوکس سیستمی است که هرچه بیشتر با آن کار کنید، قابلیت‌های جدیدتری کشف خواهید کرد.
